# TLOS
The Legend Of Souls GAME

Nome: Vinícius Simões Diniz
RA:11.00526-2
Aplicação: Jogo
Área: Integração com hardware externo
Descritivo: Este jogo é uma mistura dos conceitos e mecânicas de DARK SOULS, um jogo de RPG/AÇÃO conhecido por sua dificuldade, com o universo de THE LEGEND OF ZELDA. O jogo tem formato de plataforma 2D, onde o jogador precisa atravessar uma fase cheia de inimigos e enfrentar um BOSS (inimigo extremamente poderoso) no final da fase.

A integração com hardware externo é feita com a possibilidade de se jogar com um joystick (DUALSHOCK4)
